Code Division
Multiple Access.

CDMA (Code Division Multiple Access) is a channel access method that allows multiple users to share the same frequency band simultaneously. Unlike TDMA or FDMA, CDMA uses unique orthogonal codes to differentiate users.

User A and User B can transmit at the same time on the same frequency. Their signals mix in the air, but receivers can extract their specific data using orthogonal codes that act as mathematical filters.

This technique enables higher capacity and better security compared to traditional access methods.

How CDMA Works

Unique Codes

Each user gets a unique orthogonal code. User A has [1, 1] and User B has [1, -1]. These codes are mathematically perpendicular.

Signal Spreading

Data bits are multiplied by the user's code, creating a spread signal. This increases bandwidth but enables simultaneous transmission.

Signal Mixing

All spread signals combine in the air channel. The receiver sees a sum of all transmissions without knowing the source.

Signal Recovery

Receivers use their code as a filter. The dot product extracts their data while canceling out other users' signals.

Why CDMA Matters

CDMA revolutionized wireless communications by enabling multiple users to share the same frequency simultaneously. This increased network capacity and spectral efficiency compared to FDMA and TDMA.

Key Advantages: Better privacy (unique codes act as encryption), soft handoffs between cells (reduces dropped calls), and graceful degradation (quality decreases gradually as more users join).

Real-World Applications: CDMA formed the basis of 3G networks (WCDMA/UMTS and CDMA2000), GPS satellites, and modern 4G/5G technologies that evolved from CDMA principles.

The mathematical elegance of orthogonal codes enables what seems impossible: multiple conversations happening at once on the same frequency, perfectly separated by mathematics.
